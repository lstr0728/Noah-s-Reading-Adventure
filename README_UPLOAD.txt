# NOAH'S Reading Adventure PWA

Upload these files/folders to the root of your existing GitHub repository:

- index.html
- manifest.webmanifest
- sw.js
- icons/

After committing, wait 1-2 minutes for GitHub Pages to publish. On iPad, open the GitHub Pages link in Safari, then Share > Add to Home Screen.

This version keeps 500 words and disables Filipino browser voice so the iPad will not pronounce Filipino incorrectly.
